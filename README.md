# Fudong's Personal Website

A clean, responsive personal website hosted on GitHub Pages.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern Styling**: Contemporary design with smooth transitions and hover effects
- **Easy Navigation**: Simple navigation bar for quick access to all sections
- **Projects Showcase**: Display your projects in an attractive grid layout
- **Contact Links**: Direct links to your social media and email

## Sections

1. **Home/Hero** - Welcome message and introduction
2. **About** - Brief bio and background information
3. **Projects** - Portfolio of your work
4. **Contact** - Links to reach you via GitHub, LinkedIn, and email

## Getting Started

1. **Clone the repository** (already in your GitHub Pages directory)
2. **Edit `index.html`** - Update personal information, project details, and contact links
3. **Customize `style.css`** - Modify colors, fonts, and layout to match your brand
4. **Push to GitHub** - Commit and push changes to trigger deployment

### Google Analytics
Set your Google Analytics 4 Measurement ID in `analytics.js` by replacing `G-XXXXXXXXXX` with your real ID, for example `G-AB12C3D4E5`.

## Customization

### Colors
Edit the CSS variables at the top of `style.css`:
- `--primary-color`: Main dark color
- `--secondary-color`: Accent blue
- `--accent-color`: Highlight red

### Content
Update the text in sections:
- Hero section: Welcome message
- About section: Your bio
- Projects section: Add/edit project cards
- Contact section: Update links and email

## Deployment

GitHub Pages automatically deploys from the main branch. Simply push your changes:

```bash
git add .
git commit -m "Update website content"
git push origin main
```

Your site will be live at `https://wfudong.github.io/`
